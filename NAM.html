Нормальные алгорифмы Маркова

Входное (выходное) слово:

Правила (не более 32):


Шагов — не более 1000.

Замечание 1: особенность нормальных алгорифмов — использование дополнительных символов алфавита для управления порядком подстановок.

Замечание 2: пара «схема НАМ+входное слово» однозначно опрeделяет выходное слово.

Примеры

Пример 1. В произвольном слове P, состоящем из букв алфавита A={a, b, c}, все подряд стоящие одинаковые буквы заменить одной буквой (например, слово «abbbcaa» преобразовать в «abca»).

Пример 2. А={a,b,*}, P = *Q, где Q – произвольное слово из алфавита {a,b}. Получить Q* (переставить * в конец слова).

Пример 3. А={a,b}. Добавить в конец P символ а. (Например, из abb получить abba).

Пример 4. А={a,b,c}, требуется удвоить последнее вхождение а в P. Если а в Р не входит, не менять Р.

Пример 5. Записать слово в алфавите {a,b} в обратном порядке (например, из aababb получить bbabaa.

Задачи

1.Входит ли буква 'a' в слово.

Алфавит - {a,b,c}. Если буква 'a' входит в слово, то выдать 'a', иначе выдать пустое слово.

2.Удвоение каждой буквы

Алфавит - {a,b,c}. Удвоить каждую букву в слове.

3.Перевод числа из четверичной системы счисления в двоичную.

Алфавит - {0,1,2,3}. Перевести число из четверичной системы счисления в двоичную.

4.Удаление первой буквы

Алфавит - {a,b,c}. Если слово непустое, то удалить его первую букву; в противном случае оставить пустое слово.

5.Прибавление единицы к троичному числу

Алфавит - {0,1,2}. Прибавить единицу к троичному числу.

6.Перевод числа из единичной системы счисления в троичную

Алфавит - {|,0,1,2}. Перевести число из единичной системы счисления в троичную.

7.Раcположение вначале всех букв 'a', затем всех букв 'b', а затем всех букв 'c'

Алфавит - {a,b,c}. Переставить буквы в слове так, чтобы вначале располагались буквы 'a', затем буквы 'b', a затем буквы 'c'.

8.Содержит ли слово равное количество букв 'a' и 'b'

Алфавит - {a,b}. Если слово содержит равное количество букв 'a' и 'b', то выдать пустое слово, иначе выдать любое непустое слово.

9.Удаление правой половины слова

Алфавит - {a,b}. Слово имеет четную длину. Удалить его правую половину.

10.Удвоение слова через знак '='

Алфавит - {a,b,=}. Удвоить слово через знак '='.

11.Перевод числа из двоичной системы счисления в четверичную

Алфавит - {0,1,2,3}. Перевести двоичное число в четверичное.

12.Удаление незначащих нулей

Алфавит - {0,1,2}. Удалить из числа(троичная система) незначащие нули.

13.Перевод числа из троичной системы счисления в единичную

Алфавит - {0,1,2,|}. Перевести число из троичной системы счисления в единичную.

14.Слово abba

Алфавит - {a,b}. Если исходное слово - 'abba', то выдать букву 'a', иначе выдать пустое слово.

15.Умножение троичного числа на 2

Алфавит - {0,1,2}. Умножить троичное число на 2.

16.Деление троичного числа на 2

Алфавит - {0,1,2}. Поделить троичное число нацело на 2.

17.Переворот слова

Алфавит - {a,b}. Перевернуть слово.

18.Сложение двух троичных чисел

Алфавит - {0,1,2,+}. Сложить два троичных числа.

19.Умножение в единичной системе счисления

Алфавит - {|,*}. Перемножить 2 числа, записанных в единичной системе счисления.

20.Баланс скобок

Алфавит - {(,a,)}. Если слово сбалансировано относительно скобок, то выдать пару скобок (), а в противном случае выдать пустое слово.

21.Вычитание двух троичных чисел

Алфавит - {0,1,2,-}. Вычесть из первого троичного числа второе (считается, что второе число не больше первого).

22.Нахождение остатка от деления в единичной системе счисления

Алфавит - {|,:}. Даны 2 числа в единичной системе счисления. Найти остаток от деления первого числа на второе.
